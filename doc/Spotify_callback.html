LioranBoard Spotify Authentication (part 2)

  1. Click on Edit Commands in your Spotify INIT button.
  2. value1 = credentials
    value2 = your refresh token
    value3 = button ID to display current song name and artist (already pre-filled)
  3. Click on done and then done again.
  4. Refresh your Transmitter.
  5. Your extension should automatically connect to Spotify now.

Note: If you do not see crendetials and refresh token or it says "undefined", you did something wrong in the previous steps. Return to the page and repeat the process.
